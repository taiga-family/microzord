Here you can read how to setup your application to be inserted into another app.

If you want to make your Angular application a host of other applications, see Angular as a host

Installation

See Angular package installation

Replace the platform

You should create file with an application and expose it via module federation. Microzord provides createApp helper to convert your application to microzord child application.

Now you can build your application as usual and dist folder of your app will contain microzord.json assets map.
